# 通信

通信に関する情報。

## メソッド

| メソッド  | 主な用途 |
|-----------|----------|
| GET       | 情報を取得します。<br/>パラメータはURLパラメータを使用します。 |
| POST      | 情報を新規追加します。    |
| PUT       | 情報を修正します。        |
| DELETE    | 情報を削除します。        |

## 通信内容

フロントエンドとバックエンド間の通信は、基本的に **JSON** で行います。  
キーの命名規則は **アッパーキャメルケース** です。

例:

    {
        "KeyName": "data"
    }
